Description :
This is a simple HTML and CSS project where a user-friendly Login Page is created using Flexbox layout. The page is divided into two sections: one side displays an image, and the other side contains a login form with email and password input fields. The design is clean, responsive, and easy to use.

Technologies Used :
HTML, CSS

How to Run :
Open the index.html file in any browser.
You will see a styled Login Page with an image section and a login form.
